# Changelog

このプロジェクトの変更履歴。バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に従う。

## [0.2.0] - 2026-07-13

### Added
- ビルドキャッシュ掃除機能（`Library/` `Temp/` などの全削除。Windows のみ）。Unity 終了時に外部ヘルパーが削除する予約方式で、手作業なしに掃除できる。目立つ警告・プロジェクト規模の数値提示・削除対象の全件列挙・二段階確認・終了時の再警告・起動時の再インポート通知・所要時間の実測記録・手動実行用 `clean-cache.bat` を同梱。
- 重複アセット検出（SHA-256 で内容が完全一致するファイルを検出。レポートのみで自動削除はしない）。
- 退避前の `.unitypackage` バックアップ（任意）。
- 退避フォルダの完全削除（二段階確認つき。退避フォルダ以外は削除しない）。
- ファイル単位モード（フォルダ集約を無効化して1ファイルずつ列挙。上級者向け・既定 OFF）。
- 依存グラフのキャッシュによるスキャン高速化。

### Changed
- README を使い方ガイドとして全面刷新。
- ユーザーに見える文字列・コード内から内部設計資料の識別子を除去し、平易な表現へ統一。

## [0.1.0] - 2026-07-13

### Added
- 初回リリース（Phase 1 / MVP）。
- アバターから参照されていない未使用アセットの検出（Mark & Sweep）。
- ルート集合の指定: フォルダ登録・個別ファイル・`VRCAvatarDescriptor` 自動検出・暗黙ルート（Resources 等）。
- 保護ルール: 構造ヒューリスティック・既知ツール名・拡張子・ユーザーホワイトリスト（glob）。
- 導入単位フォルダ判定（固定深度／自動推定）。
- 退避（`.meta` 同伴・プロジェクトルート直下へ移動）と復元。
- 3タブ構成の Editor ウィンドウ、確認ダイアログ、進捗表示。
